# Eucharistic Miracles

A modern refresh to the [original Eucharistic Miracles](https://www.miracolieucaristici.org/) website created by Saint Carlo Acutis.

You can view this website [here](https://eucharistic-miracles.net).

## Running locally
Note the backend data is protected behind "auth" since I do not own the data. Please contact tom.snail.mail@gmail.com if you would like to contribute and test your changes locally.
```
npm i
npm run dev
```

## Contributions
The objective of this website is to be a community maintained version of the Eucharistic Miracles. So contributions are absolutely welcome, just clean up any AI slop should you be using that predominately.
